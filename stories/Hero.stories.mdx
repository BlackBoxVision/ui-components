# Hero

Stories related to **Hero** component.

## Example

This is how you can use **Hero** component:

```jsx
import React from 'react';
import { Hero } from '@blackbox-vision/ui-components';

export const MyHero = () => (
  <Hero
    title="We are the Agency that will accompany you in the Digital Transformation of your Company"
    subtitle="We provide creative solutions to the needs of our clients"
  />
);

MyHero.displayName = 'MyHero';
```
