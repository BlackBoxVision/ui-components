# Footer

Stories related to **Footer** component.

## Example

This is how you can use **Footer** component:

```jsx
import React from 'react';
import { Footer } from '@blackbox-vision/ui-components';

import LanguageButtons from './LanguageButtons';

export const MyFooter = () => (
  <Footer
    copyright={`BlackBox Vision @ ${new Date().getFullYear()} - All Rights Reserved`}
    LanguageButtons={<LanguageButtons />}
  />
);

MyFooter.displayName = 'MyFooter';
```
