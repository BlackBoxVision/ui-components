# Header

Stories related to **Header** component.

## Example

This is how you can use **Header** component:

```jsx
import React from 'react';
import { Header } from '@blackbox-vision/ui-components';

import LogoComponent from './LogoComponent';
import MenuComponent from './MenuComponent';

export const MyHeader = () => (
  <Header>
    <LogoComponent />
    <MenuComponent />
  </Header>
);

MyHeader.displayName = 'MyHeader';
```
