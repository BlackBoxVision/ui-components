## Example

This is how you can use **SectionCard** component:

```jsx
import React from 'react';
import { SectionCard } from '@blackbox-vision/ui-components';

export const MySectionCard = () => (
  <SectionCard
    title="We discover"
    image="https://www.blackbox-vision.tech/static/needs-e78b2aa36c17de9132482ac752cbfac8.svg"
```
